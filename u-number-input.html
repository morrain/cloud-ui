数字输入（NumberInput）

示例

基本形式

声明式

基本

<u-number-input></u-number-input>

format属性

<u-number-input format="00" value="5"></u-number-input>

autofocus

<u-number-input value="12" autofocus="autofocus"></u-number-input>

只读

<u-number-input value="12" readonly="readonly"></u-number-input>

最大值 最小值

<u-number-input value="12" min="5" max="20"></u-number-input>

指定宽高

<u-number-input value="12" width="150" height="40"></u-number-input>

禁用

<u-number-input value="12" disabled="disabled"></u-number-input>

获取改变后的值change方法

<template>
<u-number-input @change="change" :value="value" :readonly="readonly"></u-number-input>
</template>

<script>
export default {
	data: {
		value: 12,
		readonly: false,
	},
    methods: {
        change(value) {
           console.log(value.value);
        },
    },
};
</script>

API

Class

NumberInput

Extends 。

Options

new NumberInput()

ParamTypeDefaultDescription
options object = 绑定属性
value string 0 <=> 文本框的值
state string '' <=> 文本框的状态
min number => 最小值
max number => 最大值
autofocus boolean false => 是否自动获得焦点
readonly boolean false => 是否只读
disabled boolean false => 是否禁用
width string '140' => 输入框宽度
width string '34' => 输入框高度

Methods

numberInput.add(value)

调整数值

ParamTypeDefaultDescription
value number 0 加/减的值
ReturnTypeDescription
value number 计算后的值

numberInput.isOutOfRange(value)

是否超出规定的数值范围

ParamTypeDefaultDescription
value number 待测的值
ReturnTypeDescription
number boolean<br>number 如果没有超出数值范围，则返回false；如果超出数值范围，则返回范围边界的数值

Events

numberInput.$on('change')

数值改变时触发

PropertyTypeDescription
sender object 事件发送对象
value number 改变后的数值
