# 开关 Switch

## 示例
### 基本形式

``` html
<u-switch></u-switch>
```

### With Text

``` html
<u-switch with-text></u-switch>
```
### With Text

``` html
<u-switch>change</u-switch>
```
## Switch API

### Attrs/Props

| Prop | Type | Default | Description |
| --------- | ---- | ------- | ----------- |
| with-text | Boolean | `false` | 开关文字提示 |

### Slots

| Slot | Description |
| ---- | ----------- |
| (default) | 开关中显示文字 |
