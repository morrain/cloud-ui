时间选择（TimePicker）

示例

基本形式

声明式

基本

<u-time-picker width="56" time="15:46:33"></u-time-picker>

禁用

<u-time-picker width="56" time="15:46:33" disabled="disabled"></u-time-picker>

只读

<u-time-picker width="56" time="15:46:33" readonly="readonly"></u-time-picker>

日期范围

<u-time-picker time="15:46:33" width="56" min-time="12:30:00" max-time="14:45:00"></u-time-picker>

<template>
<u-time-picker width="56" :time="time" @change="change($event.time)"></u-time-picker>
</template>

<script>
export default {
	data() {
		return {
			time: '20:12:12',
		};
	},
    methods: {
        change(time) {
            console.log(time);
        },
    },
};
</script>

API

Class

TimePicker

Extends 。

Options

new TimePicker()

ParamTypeDefaultDescription
data object = 绑定属性
data.time string '00:00' <=> 当前的时间值
data.minTime string '00:00' => 最小时间
data.maxTime string '23:59' => 最大时间
data.autofocus boolean false => 是否自动获得焦点
data.readonly boolean false => 是否只读
data.disabled boolean false => 是否禁用
data.visible boolean true => 是否显示
data.class string '' => 补充class

Methods

timePicker.isOutOfRange(time)

是否超出规定的时间范围

ParamTypeDefaultDescription
time Time 待测的时间
ReturnTypeDescription
time boolean<br>Time 如果没有超出时间范围，则返回false；如果超出时间范围，则返回范围边界的时间

Events

timePicker.$on('change')

时间改变时触发

PropertyTypeDescription
sender object 事件发送对象
time object 改变后的时间
