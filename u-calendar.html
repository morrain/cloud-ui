日历（Calendar）

示例

基本形式

声明式

基本

<u-calendar date="2018-08-08"></u-calendar>

只读 readonly

<u-calendar date="2018-08-08"  readonly="readonly"></u-calendar>

最大值maxDate 最小值minDate

<u-calendar date="2018-08-08" min-date="2018-08-02" max-date="2018-08-18"></u-calendar>

禁用disabled

<u-calendar date="2018-08-08" min-date="2017-08-08" max-date="2019-08-08" disabled="disabled"></u-calendar>

API

Class

Calendar

Extends 。

Options

new Calendar()

ParamTypeDefaultDescription
options object = 绑定属性
date Date<br>string TODAY <=> 当前选择的日期
minDate Date<br>string null => 最小日期，如果为空则不限制
maxDate Date<br>string null => 最大日期，如果为空则不限制
readonly boolean false => 是否只读
disabled disabled disabled => 是否禁用

Methods

calendar.addYear(year)

调整年份

ParamTypeDefaultDescription
year number 0 加/减的年份
ReturnTypeDescription
No Returns

calendar.addMonth(month)

调整月份

ParamTypeDefaultDescription
month number 0 加/减的月份
ReturnTypeDescription
No Returns

calendar.select(date)

选择一个日期

ParamTypeDefaultDescription
date Date null 选择的日期
ReturnTypeDescription
No Returns

calendar.isOutOfRange(date)

是否超出规定的日期范围

ParamTypeDefaultDescription
date Date 待测的日期
ReturnTypeDescription
date boolean<br>Date 如果没有超出日期范围，则返回false；如果超出日期范围，则返回范围边界的日期

Events

calendar.$on('change')

日期改变时触发

PropertyTypeDescription
sender object 事件发送对象
date object 改变后的日期

calendar.$on('select')

选择某一个日期时触发

PropertyTypeDescription
sender object 事件发送对象
date object 当前选择的日期
