API

Class

Toast

Extends 。

Options

new Toast()

ParamTypeDefaultDescription
position String => 显示位置
single boolean false => 始终只显示一个提示
state String => info,success,warnning,error不同类型的提示
message String => 提示信息
visible boolean true => 显示/隐藏

Methods

toast.closeAll()

关闭所有消息

ParamTypeDefaultDescription
ReturnTypeDescription
No Returns

toast.[info|success|warning|error](message[, duration])

弹出特殊类型的消息。为show方法的简写方式。

ParamTypeDefaultDescription
message string '' 消息内容
duration number toast.duration 该条消息的停留毫秒数。如果为0，则表示消息常驻不消失。如果不填，则使用toast默认的duration。
ReturnTypeDescription
No Returns

Static Methods

Toast.show(message[, duration][, state])

弹出一个消息

ParamTypeDefaultDescription
message string '' 消息内容
duration number toast.duration 该条消息的停留毫秒数。如果为0，则表示消息常驻不消失。如果不填，则使用toast默认的duration。
state string '' 消息状态，可选参数：`info`、`success`、`warning`、`error`
ReturnTypeDescription
item object 返回弹出的消息项
