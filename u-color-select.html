颜色选择扩展（SelectColor）

示例

基本形式

方法

<template>
<u-color-select :options="options" :value="value" @select="select($event.value)"></u-color-select>
</template>

<script>
export default {
    data() {
        return {
            value: '#EE867D',
            options: [
                {
                    value: '#EE867D',
                },
                {
                    value:'#805DCF'
                },
                {
                    value:'#40CCCA'
                },
                {
                    value:'#FFCA12'
                },
                {
                    value:'#30A8DC'
                },
                {
                    value:'#E1AEF6'
                },
                {
                    value:'#D2D2D2'
                },
            ],
        };
    },
    methods: {
        select(value) {
            console.log(value);
            this.value = value;
        }
    },
};
</script>

API

Class

Select

Extends 。

Options

new Select()

ParamTypeDefaultDescription
options Array => 下拉菜单列表
readonly boolean false => 是否只读
disabled boolean false => 是否禁用
width string<br>number => 输入框长度

Events

select.$on('select')

选中列表项时触发

PropertyTypeDescription
sender object 事件发送对象
selected object 选中后的列表对象
value String 选中后的列表对象的值

select.$on('change')

选中列表项改变时触发

PropertyTypeDescription
sender object 事件发送对象
selected object 改变后的列表对象
value String 改变后的列表对象的值
